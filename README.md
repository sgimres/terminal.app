# Terminal

An interactive portfolio presented as a terminal application. Connect via SSH to explore projects, skills, and more — right from your command line.

## Connect

```
ssh eth0.top
```

> [!TIP]
> Don't forget to sign the guest book while you're there!

## Requirements

To use the terminal app in your terminal, you'll need:

- A modern Unicode/Emoji supported terminal emulator like:
  - [Windows Terminal](https://github.com/microsoft/terminal) — Windows
  - [WezTerm](https://wezfurlong.org/) — Cross-platform
  - [Alacritty](https://alacritty.org/) — Cross-platform
  - [Ghostty](https://ghostty.org/) — Linux and macOS
  - [Kitty](https://sw.kovidgoyal.net/kitty/) — Linux and macOS

- SSH client
